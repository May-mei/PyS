1.html的语法
1）什么是HTML标记语言
定义：表示网页信息的符号标记语言
特点：可以设置文本的格式
      可以创建列表
      可以插入图像和媒体
      可以建立表格
      超链接，可以使用鼠标点击超链接来实现页面之间的跳转
2）HTML的标记和他的属性
html文档的保存格式：.html/.htm/.xhtml
标记和被标记的内容构建出HTML文档：格式、内容
标记的属性
	标记的属性：就是用来控制我们的内容（图像、文本等的）如何的现实
	格式：<标记 属性1=属性值 属性2=属性值 ......>内容</标记>


3）语法不区分字母大小写
4）文档注释
注释一段内容时使用<!--"开始，以"-->结束
5）代码格式
空格键和回车键在网页中都不会起任何作用，为了让代码清晰易读，可以使用空格和回车键进行编排
注意：缩进时保持严格的规则，以“Tab”键进行缩进

2.html的基本结构
1）<html>内容</html>
解释：html文档的文档标记，也称为html开始标记
功能：这对标记分别位于网页的最前端和最后端
2）<head>内容</head>
解释：html文件头标记，也称为html头信息开始标记
功能：用来包含文件的基本信息，比如网页的标题、关键字，在<head></head>
			内可以放<title></title>、<meta></meta>、<style></style>等等标记
3）<title>内容</title>
解释：HTML文件标题标记
功能：网页的主题，显示在浏览器的窗口的左上边
		<title></title>标记中不能包含其他标记
4）<body>内容</body>
解释：html文档的主体标记
功能：<body>...</body>是网页的主体部分，在此标记之间可以包含如<p></p>
			<h1></h1>、<br>、<hr>等等标记，正是由这些内容组成了网页
body标记的常见属性
	bgcolor:设置背景颜色
	text:设置文本颜色
	link:设置连接颜色
	vlink:已经访问了的链接颜色
	alink:正在被点击的链接颜色
5）<meta>内容</meta>
解释：页面的元信息
功能：提供有关页面的元信息，比如针对搜索引擎和更新频度的描述和关键字
必要的属性：content 值：some_text  定义name属性相关的元信息
常见的属性：常见的name属性  1.author 2.keywords 3.description 4.others  把
			content属性关联到一个名称，比如描绘出网页的关键字：<meta name="keywords",
			content="麦子学院IT在线培训"
3.文档设置标记
1）格式标记
	<br>	强制换行标记  让后面的文字、图片、表格等等显示在下一行
	<p>		换段落标记
	<center>	居中对齐标记
	<pre>	预格式化标记   保留预先编排的格式
	<li>	列表项目标记   每一个列表使用一个<li>标记
	<ul>	无序列表标记
	<ol>	有序列表标记  可以显示特定的一些顺序 ：
		I)格式:<ol type="符号类型"> <li type="符号类型"></li><li type="符号类型"></li></ol>
		II)有序列表的type属性值:1）阿拉伯数字，默认type属性值
											  2）大写字母A B C等等
											  3）小写字母a b c
											  4）大写罗马数字
											  5）小写罗马数字
		III)value 指定一个新的序列数字起始值
		IV)列表可以进行嵌套
	<dl><dt><dd> 定义型列表：使用场合、格式
		<dl>
		<dt>软件说明：</dt>
		<dd>简单介绍软件的功能及基本应用</dd>
		<dt>软件界面</dt>
		<dd用于选择软件的外观></dd>
		<dl>
	<hr>	水平分割线标记
	<div>	分区显示标记，也称之为层标记
		I)	常用来编排一大段的HTML段落，也可以用于格式化表和<p>很相似
		II）层可以多层嵌套使用
2）文本标记
hn  标题标记  共有6个级别，n的范围1-6，不同级别对应显示大小不一样的标题，h1最大，h6最小
font 字体标记设置 设置字体的格式 三个常用属性
i）size（字体大小） <font size="3">取值范围是1-7
ii）color（颜色）
iii）face（字体）
